# Week 3 – Supply Chain Performance Analysis using Python & Power BI

## Project Overview

This project was completed as part of my Data Analysis Internship at LogicStack.

The objective of this project was to analyze a real-world pharmaceutical supply chain dataset using Python and Microsoft Power BI. The project involved data cleaning, exploratory data analysis (EDA), feature engineering, KPI calculation, dashboard development, and business insight generation.

---

## Objectives

* Clean and preprocess supply chain data
* Perform exploratory data analysis (EDA)
* Calculate key business KPIs
* Analyze shipment delays and freight costs
* Evaluate supplier and country performance
* Create an interactive Power BI dashboard
* Generate business recommendations

---

## Dataset Information

The dataset contains pharmaceutical shipment records, including:

* Shipment ID
* Country
* Vendor
* Shipment Mode
* Product Group
* Brand
* Manufacturing Site
* Scheduled Delivery Date
* Delivered to Client Date
* Delivery Recorded Date
* Freight Cost (USD)
* Line Item Value
* Delivery Delay

Total Records: **10,324**

---

## Data Cleaning

The following preprocessing steps were completed using Python:

* Missing value handling
* Date conversion
* Duplicate checking
* Data type correction
* Feature engineering
* Created Delivery Delay column
* Exported cleaned dataset
